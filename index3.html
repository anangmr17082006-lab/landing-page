Developer First

SQL, But Faster.

Mendukung SQL standard dengan ekstensi time-series. Kompatibel dengan PostgreSQL wire protocol.

  • Time-bucket aggregation (1s, 1m, 1h)
  • Window functions optimized for Gap Filling
  • Continuous Aggregates
analytics.sql 
SELECT 
    time_bucket('1 minute', created_at) AS one_min,
    device_id,
    AVG(cpu_usage) AS avg_cpu,
    PREDICT_LINEAR(cpu_usage, '1 hour') AS forecast
FROM server_metrics
WHERE created_at > NOW() - INTERVAL '6 hours'
GROUP BY one_min, device_id;
