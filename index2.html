Integrasi Seamless SDK

Gunakan Python SDK kami untuk mendaftarkan model langsung dari notebook eksperimen Anda. Versi dikelola secara otomatis.

  • Support Scikit-learn, TensorFlow, PyTorch
  • Manajemen Environment Otomatis
  • Built-in A/B Testing Routing
deploy_script.py
import modelforge as mf

# Inisialisasi Project
project = mf.get_project("fraud-detection")

# Log dan Deploy Model
deployment = project.deploy(
    model_path="models/xgboost_v2.pkl",
    strategy="canary",       # Canary Deployment 10%
    drift_check=True,        # Aktifkan auto-monitoring
    min_instances=2          # Auto-scaling min replicas
)

print(f"Model live at: {deployment.endpoint}")
